# Mio

Stands for Make it out (of the gc)!

This is a temporary calendar that you can use to plan temporarily, and just delete and forget about the calendar by just deleting the tab. I was once planning for a trip, and I was looking for a temporary calendar onine and didn't find one?? So I decided to create one myself! I wanted to keep it very simple without anything distracting so it's as easy to use as possible since it is just a small tool to help plan.

I also used this to patch up on some of my JS and learn stuff about doing stuff with dates.
