# Mio

Stands for Make it out (of the gc)!

This is a temporary calendar that you can use to plan temporarily and send it to your friends / family as well. I was once planning for a trip, and I was looking for a calendar onine and didn't find one?? So I decided to create one myself!

Add
- switch between months
- hold and drag for event spanning multiple days
- customize events (eg repeating weekly)
